# Memento Curated 💛

Memento Curated is an elegant, premium web portal built for a boutique jewelry brand. Our mission is to prove that luxury does not have to be out of reach by delivering carefully handpicked, stylish, and timeless jewelry pieces that balance quality, elegance, and affordability.

## Brand Mission

> Jewelry is more than just an accessory—it's a way to express yourself and celebrate life's special moments. At Memento Curated, we carefully handpick stylish and timeless pieces that offer the perfect balance of quality, elegance, and affordability. Luxury doesn't have to be out of reach, and we're here to prove it.

---

## Design Philosophy

Adhering to high-agency design guidelines, the interface is styled with a luxury-oriented aesthetic:
* **Color Palette**: Deep Obsidian/Charcoal backdrop (`#09090b`) with refined gold accents (`#d4af37`) mirroring the brand's logo.
* **Layout Structure**: Asymmetric, split-screen Hero section pairing left-aligned bold typography with the brand's transparent logo showcase under radial ambient lighting on the right.
* **Liquid Glass Refraction**: Bento-grid inspired features and product cards containing subtle `1px` translucent borders (`border-white/10`) and inner-refractive shadows to evoke depth.
* **Zero Emojis in Code**: Pure, professional code structures relying strictly on premium design layouts and high-end typography (Geist + Playfair style layout pairings).

---

## Key Features

1. **Elegant Catalog Grid**: Showcases curated pieces with responsive hover states, smooth image zooming, and category badge identifiers.
2. **Interactive Catalog Filtering**: Allows clients to easily sort jewelry by category (Rings, Necklaces, Bracelets, Earrings) on-the-fly.
3. **Dynamic Client Preview (Add Piece)**: A built-in, functional playground for the client to preview how new pieces will look when added to the collections. Using a tactile modal panel, clients can publish pieces which immediately populate the catalog in real-time.

---

## Tech Stack

* **Core**: Next.js 16.2 (Turbopack) & React 19
* **Styling**: Tailwind CSS v4 (using PostCSS integration)
* **Language**: TypeScript
